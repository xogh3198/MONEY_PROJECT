## 금융 계산 규칙 (절대 위반 금지)

### 세율 규정 (2024년 기준)
- 배당소득 원천징수: 15.4% (소득세 14% + 지방소득세 1.4%)
- ISA 비과세 한도: 일반형 200만원, 서민형 400만원
- ISA 비과세 초과분: 9.9% 분리과세
- IRP: 퇴직 시점까지 과세이연, 수령 시 연금소득세 3.3~5.5%
- 해외주식 배당: 현지 원천징수 15% (미국 기준), 국내 추가 과세 없음
- 금융소득종합과세: 연간 금융소득 2,000만원 초과 시 종합과세

### 금액 계산 필수 규칙
1. **float 타입 사용 절대 금지** → Go: shopspring/decimal, Python: decimal.Decimal
2. **세율 하드코딩 금지** → config/tax_rates.yaml에서 로드
3. **반올림**: 원 단위 이하 절사 (한국 금융 관행, math.Floor)
4. **모든 계산 함수에 테이블 드리븐 테스트** 작성
5. **경계값 테스트 필수**: 비과세 한도 직전(199만), 한도(200만), 초과(201만)

### ISA 계좌 규칙
- 연간 납입 한도: 2,000만원
- 의무 가입 기간: 3년 (중도 해지 시 세제 혜택 소멸)
- 비과세 한도: 일반형 200만원, 서민형/농어민 400만원
- 한도 초과 시: 초과분에 9.9% 분리과세 (종합과세 합산 안 됨)
- 손익통산: ISA 내 이익과 손실을 합산하여 순이익에만 과세

### IRP 계좌 규칙
- 연간 납입 한도: 1,800만원 (DC형 퇴직연금 포함)
- 세액공제: 납입액의 13.2~16.5% (총급여 5,500만원 기준 구분)
- 배당소득: 수령 시점까지 과세이연 (복리 효과)
- 중도 인출: 제한적 (무주택자 주택 구입 등 예외)

### 배당락일 규칙
- 배당기준일: 주주명부에 등재된 주주에게 배당 지급
- 배당락일: 배당기준일 1영업일 전 (T+2 결제 기준)
- 배당락일 당일 매수: 배당금 수령 **불가**
- 배당락일 전일(영업일)까지 매수 완료: 배당금 수령 가능
- 공휴일/주말 고려한 **영업일 계산** 로직 필수

### 코드 작성 시 주의사항
- 세율 변경 가능성 고려 → 설정 파일로 외부화
- 계좌 유형(일반/ISA/IRP) 구분 로직 필수
- 국내/해외 주식 배당 세율 차이 처리
- 원화/외화 환산 시 환율 적용 시점 명시
