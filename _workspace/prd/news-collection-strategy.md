# 뉴스 수집 전략

## 현실적 소스 분석

### 무료로 사용 가능한 경제 뉴스 소스

| 소스 | 방법 | 데이터 | 제한 |
|------|------|--------|------|
| **네이버 뉴스 검색 API** | REST API (개발자 등록) | 제목, 요약, 링크, 날짜 | 하루 25,000건 |
| **한경 RSS** | XML 파싱 | 제목, 본문 일부 | 최신 20건 |
| **매경 RSS** | XML 파싱 | 제목, 요약 | 최신 20건 |
| **연합뉴스 RSS** | XML 파싱 | 제목, 요약, 이미지 | 최신 30건 |
| **Google News RSS** | XML 파싱 | 제목, 출처, 링크 | 무제한 |

### 추천 전략: 네이버 뉴스 API (주력) + RSS (보조)

**이유:**
- 네이버 API는 검색 키워드 기반 → 카테고리별 정확한 수집 가능
- 하루 25,000건이면 분당 17건 → 충분
- JSON 응답이라 파싱 간편 (RSS XML보다 안정적)
- 한국 경제 뉴스 커버리지 최고

### 수집 플로우
```
[15분마다 실행]
1. 네이버 뉴스 API 호출 (카테고리별 키워드)
   - "코스피 증시" → DOMESTIC
   - "나스닥 미국증시" → OVERSEAS
   - "원달러 환율" → FOREX
   - "기준금리" → RATE
   - "비트코인" → CRYPTO
2. 중복 체크 (URL 기반)
3. 감성 분석 (키워드 기반 → 추후 LLM 교체)
4. DB 저장
5. RSS 소스 보조 수집 (네이버에 없는 전문 기사)
```

### 네이버 API 키 필요
- 등록: https://developers.naver.com
- Client ID + Client Secret 발급 (무료)
- 환경변수: NAVER_CLIENT_ID, NAVER_CLIENT_SECRET
