# 프로젝트 확정 결정 사항

## 기술 스택 결정 (2026-06-17)
| 항목 | 결정 | 사유 |
|------|------|------|
| AI Agent | Kiro (Claude Opus 4.6) | IDE 통합 개발 |

## 아키텍처 결정

- Kafka로 서비스 간 비동기 통신
- MinIO로 API 응답 원본 캐싱
- 하네스 엔지니어링 적용 (revfactory/harness 기반)
